### Reporting Bugs

If you find a bug, please open a [new Issue](https://github.com/H3m0rv1ch1/enjaz/issues) with:
- Clear description of the problem
- Steps to reproduce the issue
- Screenshots if possible
- System information (OS, browser version, etc.)

### Suggesting New Features

To suggest a new feature:
1. Check that no similar suggestion exists in Issues
2. Open a new Issue with "Feature Request" label
3. Explain the proposed feature and its benefits

### Contributing Code

1. **Fork the Project**
   ```bash
   git clone https://github.com/YOUR_USERNAME/enjaz.git
   ```

2. **Create a New Branch**
   ```bash
   git checkout -b feature/your-feature-name
   ```

3. **Make Changes**
   - Follow existing code standards
   - Add clear comments
   - Ensure code works correctly

4. **Test Changes**
   ```bash
   npm run dev
   npm run tauri:dev
   ```

5. **Commit Changes**
   ```bash
   git add .
   git commit -m "Add: clear description of the change"
   ```

6. **Push to GitHub**
   ```bash
   git push origin feature/your-feature-name
   ```

7. **Create Pull Request**
   - Go to the project page on GitHub
   - Click "New Pull Request"
   - Write a clear description of the changes
